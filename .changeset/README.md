# Changesets

This folder is managed by [`@changesets/cli`](https://github.com/changesets/changesets).

Each `.md` file (other than this README and AGENTS.md) represents a pending
version bump. When ready to release, run `bun run version-packages` to consume
them, bump versions, and generate changelogs.

See [AGENTS.md](./AGENTS.md) for detailed formatting rules and workflow.
